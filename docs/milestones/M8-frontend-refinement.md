# M8: 侧边栏重构 + 资源连接 + 仪表盘

## Context

M0-M7 完成了后端全部功能和前端基础页面。但前端侧边栏与原型差距很大，资源只能新增不能点击连接。本里程碑聚焦最重要的两个改进：侧边栏重构和资源连接。

## 产品边界

**做什么：**
- 侧边栏按原型重构：搜索框 + 导航 + 环境资源树（可展开/折叠）
- 资源点击连接（根据协议路由到终端/SQL/文件页面）
- 侧边栏折叠/展开（桌面端，localStorage 持久化）
- 移动端汉堡菜单
- 仪表盘快速连接 + 最近使用区域
- 环境详情页资源可点击连接

**不做什么：**
- 工作区分屏
- 审计日志页面
- 设置页详细设置区块（外观/终端/安全）
- SQL 全局查询 / AI 助手
- 终端 SFTP 面板
- 文件传输队列面板
- 收藏功能

## 子任务 8.2：资源点击连接

### 功能目标

侧边栏资源项和环境详情页资源行点击后根据协议路由。

### 路由分发

```typescript
function connectToResource(resource: { id: string; protocol: string; name: string }) {
  switch (resource.protocol) {
    case 'ssh': router.push(`/terminal/${resource.id}`); break
    case 'mysql': case 'postgresql': router.push(`/sql/${resource.id}`); break
    case 'sftp': router.push(`/files/${resource.id}`); break
    default: break
  }
  // 记录到最近使用
  addToRecent(resource)
}
```

### 提交信息

```

```
