## 子任务 10.1：审计日志页面

### 功能目标

实现审计日志专用查询页面，包含筛选条件、统计卡片、日志表格（含行展开详情）、CSV 导出。

### 修改文件

```text
packages/rex-console-web/src/
├── pages/AuditLog.vue      新增：审计日志页面
├── i18n/zh.ts, en.ts        修改：添加审计日志 i18n keys
```

### 接口设计

后端已有 `GET /api/audit-log` 端点，返回格式：
```json
{
  "data": [
    {
      "id": "log_xxx",
      "action": "resource_create",
      "result": "success",
      "summary": "在环境 env_001 中创建资源「云服务器」",
      "environment_id": "env_001",
      "resource_id": "res_xxx",
      "user_ip": "192.168.1.100",
      "created_at": "2026-06-20T10:30:00Z"
    }
  ]
}
```

### 前端交互

参考原型 `prototype/audit-log.html`：

1. **筛选条件栏**：时间范围（最近 1 小时 / 24 小时 / 7 天 / 30 天）、用户（全部 / admin）、环境（全部 / 动态加载）、操作类型（全部 / 连接 / 断开 / SQL 查询 / 文件上传 / 文件下载 / SSH 命令 / 登录）、重置按钮
2. **统计卡片**：总操作数、成功数、失败数、活跃用户数（共 4 个）
3. **日志表格**：时间（monospace）、用户、环境（带状态点）、操作（彩色标签）、摘要、结果（成功✓/失败✕）
4. **行展开详情**：点击行展开详情面板，深色背景，含结构化详情网格（协议、目标、用户、耗时、传输量等）+ 代码块展示完整命令或 SQL 语句
5. **CSV 导出按钮**
6. **分页**：页码按钮

### 操作类型颜色编码

| 操作类型 | 颜色 |
|----------|------|
| 连接 | `#3FB950` |
| 断开 | `#8B949E` |
| SQL 查询 | `#58A6FF` |
| 文件上传 | `#8B5CF6` |
| 文件下载 | `#3FB950` |
| 文件删除 | `#F85149` |
| 登录 | `#E8912D` |
| SSH 命令 | `#D29922` |

### 测试标准

- 页面能正常加载
- 四个筛选条件生效
- 统计数字正确（4 个卡片）
- 点击行展开详情面板
- CSV 导出能下载文件
- 分页工作正常

## 子任务 10.4：设置页安全设置

### 功能目标

设置页增加安全设置区块：会话超时、配置加密开关、审计日志开关（含开关联动行为）、查看审计日志跳转链接。

### 修改文件

```text
packages/rex-console-web/src/
├── pages/Settings.vue       修改：添加安全设置区块
├── i18n/zh.ts, en.ts        修改：添加安全设置 i18n keys
```

### 前端交互

参考原型 `prototype/settings.html`：

1. **会话超时**：下拉选择（15 分钟 / 30 分钟 / 1 小时 / 永不），默认 30 分钟
2. **配置加密**：开/关 toggle，默认开
3. **操作审计日志**：开/关 toggle，默认开
   - 开启时：记录所有操作，侧边栏显示审计日志入口
   - 关闭时：隐藏入口并停止记录
4. **查看审计日志**：跳转链接「查看 →」，仅审计日志开启时显示，点击跳转到审计日志页面

### 测试标准

- 安全设置能保存到 localStorage
- 会话超时设置在前端生效（控制 token 过期跳转）
- 审计日志开关切换时，侧边栏审计日志入口联动显示/隐藏
- 查看审计日志链接仅在审计开启时显示

### 提交信息

```
feat: add security settings to settings page
```
