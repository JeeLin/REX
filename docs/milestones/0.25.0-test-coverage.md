- **文件结构**：
  - 修改：`crates/rex-hub/src/auth.rs`（补充 token 生成/验证边界测试）
  - 修改：`crates/rex-hub/src/acme.rs`（补充 determine_tls_mode 各分支测试）
  - 修改：`crates/rex-hub/src/config.rs`（补充环境变量覆盖、CLI 覆盖测试）
  - 修改：`crates/rex-hub/src/settings.rs`（补充 TLS 状态查询测试）
  - 修改：`crates/rex-hub/src/self_signed.rs`（补充证书生成/加载测试）
  - 修改：`crates/rex-hub/src/audit.rs`（补充审计日志写入/查询测试）
