## 子任务详细设计

### 1 分屏拖拽修复

- **功能目标**：修复分屏布局和标签拖拽到面板的功能
- **问题描述**：分屏和拖拽基本没法用
- **文件结构**：
  - 修改：`packages/rex-console-web/src/pages/Workspace.vue`
  - 修改：`packages/rex-console-web/src/features/workspace/` 相关组件
- **接口设计**：无 API 改动，纯前端修复
- **交互设计**：分屏布局切换正常，标签可拖拽到目标面板
- **后端流程**：无
- **测试标准**：
  - Alt+1~5 切换布局正常
  - 标签拖拽到空面板自动填充
  - 标签拖拽到已占用面板交换内容
- **提交信息**：`fix: workspace split screen and drag-and-drop`

### 2 SSH 终端复制粘贴修复

- **功能目标**：修复 SSH 终端复制粘贴功能，解决与浏览器快捷键冲突
- **问题描述**：Ctrl+C/V 与浏览器冲突，复制粘贴没法用
- **文件结构**：
  - 修改：`packages/rex-console-web/src/features/terminal/` 相关组件
- **接口设计**：无 API 改动
- **交互设计**：
  - Ctrl+Shift+C 复制选中内容（不与浏览器冲突）
  - Ctrl+Shift+V 粘贴
  - 右键菜单复制/粘贴正常工作
- **后端流程**：无
- **测试标准**：
  - 终端选中文本后 Ctrl+Shift+C 复制到剪贴板
  - Ctrl+Shift+V 从剪贴板粘贴到终端
  - 右键菜单复制/粘贴正常
- **提交信息**：`fix: terminal copy/paste with Ctrl+Shift+C/V`

### 3 SQLite 连接问题修复

- **功能目标**：修复 SQLite 连接错误不显示和打开不自动连接的问题
- **问题描述**：
  - 连接失败时页面没显示错误（`{"type":"error","message":"SQLite 连接失败: unable to open database file: /app/data/hub.db"}`）
  - SQLite 打开没有自动连接
- **文件结构**：
  - 修改：`packages/rex-console-web/src/features/sql/` 相关组件
  - 修改：`crates/rex-hub/src/sqlite.rs`（如需后端调整）
- **接口设计**：无 API 改动，前端错误处理修复
- **交互设计**：
  - 连接失败时显示错误 Toast 或错误状态
  - 打开 SQLite 资源时自动建立连接
- **后端流程**：无
- **测试标准**：
  - 连接失败时页面显示错误信息
  - 打开 SQLite 资源自动连接
- **提交信息**：`fix: SQLite connection error display and auto-connect`

### 4 仪表盘与审计日志修复

- **交互设计**：

  - 审计日志操作列显示中文
- **后端流程**：无
- **测试标准**：
