# 0.33.0: SQL 编辑器与结果菜单补全

## Context

0.32.1 完成了 Bug 修复与体验修复。本阶段聚焦 SQL 控制台的功能补全，主要是右键菜单的完善和导出功能的增强。

版本类型：minor（新功能）

## 产品边界

**做什么：**
- SQL 编辑器右键菜单补全（执行选中/全部 SQL、格式化、大小写转换、注释/取消注释、保存、插入模板、历史记录）
- SQL 结果表格右键菜单补全（复制行/单元格/列/JSON、排序、导出、生成 UPDATE/DELETE）
- 查询标签右键菜单补全（关闭、保存、重命名、复制 SQL、执行 SQL）
- 导出功能增强（支持 CSV/JSON 导出）

**不做什么：**
- 执行计划功能（需要后端 EXPLAIN 支持，推到后续里程碑）
- 库表结构树的导出表数据、新建表（需要新后端 API，推到后续里程碑）

## 子任务详细设计

### 1 SQL 编辑器右键菜单补全

- **功能目标**：完善 SQL 编辑器的右键上下文菜单，提升编辑效率
- **文件结构**：
  - 修改：`packages/rex-console-web/src/features/sql/SqlEditor.vue`（右键菜单）
  - 修改：`packages/rex-console-web/src/i18n/zh.ts`（i18n）
  - 修改：`packages/rex-console-web/src/i18n/en.ts`
- **接口设计**：无 API 改动，纯前端
- **交互设计**：
  右键菜单项：
  - 执行选中 SQL（仅选中时可用）
  - 执行全部 SQL
  - ─── 分隔线 ───
  - 剪切（仅选中时可用）
  - 复制（仅选中时可用）
  - 粘贴
  - ─── 分隔线 ───
  - 格式化 SQL
  - 大小写转换 ▸（全部大写、全部小写、首字母大写）
  - 注释/取消注释
  - ─── 分隔线 ───
  - 保存为查询文件
  - 插入模板 ▸（SELECT、INSERT、UPDATE、DELETE、CREATE TABLE）
  - ─── 分隔线 ───
  - 历史记录
- **后端流程**：无
- **测试标准**：
  - 右键菜单显示所有菜单项
  - 选中文本时，执行选中、剪切、复制项可用
  - 无选中文本时，执行选中、剪切、复制项禁用
  - 格式化 SQL 功能正常
  - 大小写转换功能正常
  - 注释/取消注释功能正常
  - 保存功能正常
  - 插入模板功能正常
  - 历史记录功能正常
- **提交信息**：`feat: complete SQL editor context menu`
