# 0.4.0: 设置项连接实际行为

## Context

v0.3.0 完成了 UI 打磨与 i18n 补全。当前 Settings 页面的终端设置（字体大小、字体族、光标闪烁、保活间隔）和安全设置（会话超时、配置加密、审计日志开关）的 UI 已实现，值存储在 localStorage，但未连接到实际的终端组件和安全行为。用户修改设置后看不到任何效果。

本阶段将这些设置项连接到实际行为，使 Settings 页面真正生效。

版本类型：minor

## 产品边界

**做什么：**
- 终端设置影响 SSH 终端的实际显示（字体大小、字体族、光标闪烁）
- SSH 保活间隔设置传递到后端 SSH 连接
- 会话超时设置在前端执行自动登出
- 审计日志开关控制前端是否显示审计日志入口

**不做什么：**
- 不实现配置加密开关的后端切换（当前已默认启用 AES-256 加密，切换需要重新加密/解密所有凭据，风险较高）
- 不添加新的协议支持（Redis、Docker 等留到后续里程碑）
- 不重构设置存储机制（仍用 localStorage + 事件通信）

## 子任务详细设计

### 1 终端设置连接实际行为

**功能目标：**
将 Settings 页面的终端设置（字体大小、字体族、光标闪烁、保活间隔）传递到 SSH 终端组件，使其实际生效。

**文件结构：**
```text
packages/rex-console-web/src/
├── stores/settings.ts                         修改：导出设置响应式状态
├── features/terminal/Terminal.vue             修改：读取设置并应用
├── features/workspace/panels/WorkspaceTerminal.vue  修改：读取设置并应用
├── features/settings/TerminalSection.vue      修改：使用共享设置 store
```

**接口设计：**
新增 `stores/settings.ts` 共享设置 store：
```typescript
// 终端设置
export const terminalSettings = reactive({
  fontSize: Number(localStorage.getItem('rex-term-font-size')) || 13,
  fontFamily: localStorage.getItem('rex-term-font-family') || 'JetBrains Mono',
  cursorBlink: localStorage.getItem('rex-term-cursor-blink') !== 'false',
  keepalive: Number(localStorage.getItem('rex-term-keepalive')) || 60,
})

// 更新函数（同时写入 localStorage）
export function updateTerminalSetting(key: keyof typeof terminalSettings, value: unknown) { ... }
```

终端组件通过 `watch` 监听设置变化，动态更新 xterm 实例：
- `fontSize` → `terminal.options.fontSize`
- `fontFamily` → `terminal.options.fontFamily`
- `cursorBlink` → `terminal.options.cursorBlink`
- `keepalive` → 通过 WebSocket 消息传递到后端 SSH session

**交互设计：**
- 修改字体大小后，所有打开的终端标签实时更新
- 修改保活间隔后，新建立的 SSH 连接使用新间隔

**测试标准：**
- 修改字体大小后终端立即变化
- 修改字体族后终端立即变化
- 开关光标闪烁后终端立即变化
- 保活间隔设置传递到新连接

### 2 安全设置连接实际行为

**功能目标：**
将 Settings 页面的安全设置（会话超时、审计日志开关）连接到实际行为。

**文件结构：**
```text
packages/rex-console-web/src/
├── stores/settings.ts                         修改：添加安全设置
├── composables/useSessionTimeout.ts           新建：会话超时 composable
├── features/settings/SecuritySection.vue      修改：使用共享设置 store
├── layouts/AppLayout.vue                      修改：集成会话超时
├── features/settings/AppearanceSection.vue    修改：侧边栏折叠设置
```

**接口设计：**
扩展 `stores/settings.ts`：
```typescript
export const securitySettings = reactive({
  sessionTimeout: Number(localStorage.getItem('rex-session-timeout')) || 30,
  auditEnabled: localStorage.getItem('rex-audit-enabled') !== 'false',
})
```

新增 `composables/useSessionTimeout.ts`：
```typescript
// 监听用户活动（鼠标、键盘、触摸）
// 超时后自动跳转到 /login 并清除 token
// 支持 pause/resume（如弹窗打开时暂停计时）
```

**交互设计：**
- 用户无操作 30 分钟（默认）后自动登出，跳转到登录页
- 任何鼠标移动、键盘输入、触摸操作重置计时器
- 审计日志开关关闭时，侧边栏隐藏审计日志入口

**测试标准：**
- 设置会话超时为 15 分钟后，15 分钟无操作自动登出
- 审计日志开关关闭后侧边栏不显示审计日志入口
- 审计日志开关打开后侧边栏显示审计日志入口
