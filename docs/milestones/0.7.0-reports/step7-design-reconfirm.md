### 2. 设计变更说明

**变更 1：自签名不自动启用**

里程碑文档描述"无 ACME 配置 → 自签名模式"，但实际实现改为"无 TLS 配置 → HTTP only"。

原因：用户明确指出"应该是需要在设置里打开 HTTPS 配置才走 HTTPS 逻辑"。自签名不应作为默认行为，用户需要显式启用 TLS。

影响：
- `TlsMode::SelfSigned` 变为死代码（无配置路径可达）
- 默认行为从"总是 HTTPS"改为"默认 HTTP"
- 这是功能简化，不引入新问题
