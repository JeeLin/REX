# 0.14.0: 核心面板精化

## Context

0.13.0 完成了 MySQL/PostgreSQL 真实连接实现，8 种协议全部具备真实连接能力。0.12.0 完成了代码质量清理。现在各协议面板的基础功能已完整，但与 PRODUCT.md 描述的完整交互仍有差距。本里程碑聚焦于 SQL 控制台、仪表盘和全局连接菜单三个高频使用场景的精化。

版本类型：minor（功能增强）

## 产品边界

**做什么：**
- SQL 控制台：查询结果导出（JSON/CSV）+ 结果表格右键菜单
- 仪表盘：环境卡片资源类型统计徽章
- 全局连接菜单（Ctrl+N）：搜索+按环境分组的快速连接入口

**不做什么：**
- 不新增协议或资源类型
- 不修改后端 API 结构
- 不引入 AI 助手或全局查询
- 不做 SSH 终端或 SFTP 文件管理的大改动

## 子任务详细设计

### 1 SQL 查询结果导出

**功能目标：**
在 SQL 控制台结果区添加导出按钮，支持将查询结果导出为 JSON 和 CSV 格式下载。

**文件结构：**
```text
packages/rex-console-web/src/
├── features/workspace/panels/WorkspaceSql.vue   修改：添加导出按钮和逻辑
└── api/sql.ts                                    不修改（数据已在前端）
```

**接口设计：**
无需后端 API，纯前端导出：
- JSON 导出：将 `SqlResult.rows` + `SqlResult.columns` 序列化为 JSON 文件
- CSV 导出：将列名作为表头，行数据用逗号分隔，处理引号转义

**交互设计：**
参考 PRODUCT.md §3.7 结果区：
- 结果区顶部工具栏添加导出按钮组：📋 复制 · ⬇ 导出（下拉：JSON / CSV）
- 点击后触发浏览器下载，文件名格式：`{resource_name}_{database}_{timestamp}.{ext}`

**提交信息：**
```
feat: add SQL query result export (JSON/CSV)
```

### 2 SQL 结果表格右键菜单

**功能目标：**
在 SQL 查询结果表格上添加右键上下文菜单，提供快捷操作。

**文件结构：**
```text
packages/rex-console-web/src/features/workspace/panels/WorkspaceSql.vue   修改：添加右键菜单
```

**交互设计：**
参考 PRODUCT.md §3.7 查询结果表格右键菜单，实现以下核心项：

| 菜单项 | 说明 |
|--------|------|
| 复制行 | 复制当前行数据（Tab 分隔） |
| 复制单元格 | 复制当前单元格值 |
| ─── 分隔线 ─── | |
| 按此列升序排列 | 以当前列为键升序排序 |
| 按此列降序排列 | 以当前列为键降序排序 |

不实现的高级项（后续里程碑）：
- 导出当前行、生成 UPDATE/DELETE 模板

**提交信息：**
```
feat: add SQL result table context menu
```

### 3 仪表盘环境卡片资源类型徽章

**功能目标：**
在仪表盘的环境卡片中显示该环境下的资源类型分布统计。

**文件结构：**
```text
packages/rex-console-web/src/pages/Dashboard.vue   修改：添加徽章渲染
```

**交互设计：**
参考 PRODUCT.md §3.2 环境卡片网格：
- 每个环境卡片显示资源类型统计：`SSH ×2  MySQL ×1`
- 徽章使用协议色半透明背景 + 对应颜色文字（与侧边栏一致）
- 如果环境无资源则不显示

**数据来源：**
已有 `listEnvsWithResources` API 返回环境+资源列表，前端按 protocol 分组计数。

**提交信息：**
```

```

### 4 全局连接菜单（Ctrl+N）

**功能目标：**
实现工作空间的全局连接菜单，支持搜索和按环境分组的快速连接。

**文件结构：**
```text
packages/rex-console-web/src/features/workspace/Workspace.vue   修改：添加连接菜单组件
packages/rex-console-web/src/features/workspace/ConnectMenu.vue  新增：连接菜单组件
```

**交互设计：**
参考 PRODUCT.md §3.5 连接菜单：
- `Ctrl+N` 或点击 `+` 打开弹出式全局搜索菜单
- 搜索框，支持名称、地址、协议模糊搜索
- 所有可用资源**按环境分组**展示
- 每个资源项：彩色协议图标 + 名称 + 地址 + 协议标签
- 键盘导航：↑↓ 选择、Enter 连接、Esc 关闭
- 底部提示操作方式
- 点击资源项后在工作空间打开对应标签页

**数据来源：**
已有 `listEnvsWithResources` API，前端缓存并过滤。

**提交信息：**
```
feat: add global connect menu (Ctrl+N)
```
