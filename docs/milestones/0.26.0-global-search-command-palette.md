# 0.26.0: 全局搜索与命令面板

## Context

0.25.0 完成了测试覆盖率提升，项目质量基础已夯实。PRODUCT.md §3.5 明确定义了 Ctrl+K 全局搜索/命令面板功能，但代码库中尚未实现。本里程碑补齐该核心交互入口，提升工作流效率。

版本类型：minor（新增功能，向后兼容），版本号 0.26.0。

## 产品边界

**做什么：**
- 实现 Ctrl+K 快捷键唤起全局命令面板
- 面板内支持模糊搜索资源、环境、动作
- 键盘导航（上下选择、Enter 执行、Esc 关闭）
- 搜索结果按类别分组
- 面板支持鼠标点击选择

**不做什么：**
- 不修改现有资源/环境数据结构
- 不引入新的 API 端点（纯前端交互）
- 不改变现有快捷键行为

## 子任务详细设计

### 1 创建 CommandPalette 组件与搜索逻辑

- **功能目标**：实现全局命令面板 UI，支持模糊搜索、分组展示、键盘导航。
- **文件结构**：
  - 创建：`packages/rex-console-web/src/components/CommandPalette.vue`
- **接口设计**：
  - Props：`visible: boolean`
  - Events：`close`、`select(item: CommandItem)`
  - CommandItem：`{ id, label, category, icon, shortcut, action }`
- **交互设计**：
  - 模态弹窗居中，半透明黑色遮罩 + 模糊背景
  - 顶部搜索输入框，自动聚焦
  - 结果列表按类别分组（资源、动作、导航）
  - 选中项高亮，支持滚动
  - 底部提示：上下箭头导航 · Enter 选择 · Esc 关闭
  - 搜索无结果时显示空状态提示
  - 键盘导航循环（到底部再按下回到顶部）
  - 输入即时搜索，无延迟
- **搜索逻辑**：
  - 匹配资源名称、地址、协议、环境名称
  - 支持拼音首字母或模糊匹配（如 "ssh" 匹配所有 SSH 协议资源）
  - 匹配高亮显示
- **命令来源**：
  - 资源列表：从已加载的资源数据中读取
  - 导航动作：工作区、仪表盘、环境、Agent、设置
  - 操作动作：新建连接、切换布局模式、新建环境、新建资源
- **测试标准**：组件渲染、搜索过滤、键盘导航、选择执行。
- **提交信息**：`feat: add CommandPalette component with fuzzy search`

### 2 集成到 Workspace 页面并测试验证

- **功能目标**：在工作空间页面集成命令面板，监听 Ctrl+K 快捷键，端到端测试。
- **文件结构**：
  - 修改：`packages/rex-console-web/src/pages/Workspace.vue`
  - 创建：`packages/rex-console-web/src/components/__tests__/CommandPalette.test.ts`
- **接口设计**：
  - 在 Workspace 中添加 `showCommandPalette` ref
  - 全局 keydown 监听 Ctrl+K / Meta+K 打开面板
  - 面板 select 事件处理：
    - 资源类 → 调用现有 openTab 逻辑
    - 导航类 → router.push
    - 操作类 → 触发对应动作
- **交互设计**：
  - Ctrl+K 打开，Esc 关闭
  - 已在面板打开时 Ctrl+K 关闭（切换）
  - 选择资源 → 在工作空间打开对应标签并聚焦
  - 选择导航 → 跳转到目标页面
- **测试标准**：
  - 快捷键触发面板打开/关闭
  - 搜索过滤正确
  - 资源选择打开正确标签
  - 导航跳转正确
  - Esc 关闭面板
- **提交信息**：`feat: integrate CommandPalette into Workspace with Ctrl+K`
